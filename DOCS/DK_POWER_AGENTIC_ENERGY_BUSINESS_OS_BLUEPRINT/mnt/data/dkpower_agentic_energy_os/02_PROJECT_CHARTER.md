# Project Charter
# DK Power Agentic Energy Business OS

## 1. Project Purpose

Membangun sistem terpadu berbasis cloud dan AI untuk mempercepat proses penjualan, estimasi, proposal, dan pengelolaan proyek DK Power.

## 2. Business Case

Saat bisnis energy solution tumbuh, proses manual akan menjadi bottleneck. Sistem ini membantu DK Power menjual lebih cepat, menghitung lebih akurat, dan mengelola pipeline secara transparan.

## 3. Project Objectives

- Membuat MVP operasional dalam 12 minggu.
- Mengotomatisasi proposal solar panel.
- Mengintegrasikan lead dari website, simulator, dan WhatsApp.
- Menyediakan dashboard owner.
- Menyiapkan fondasi Agentic AI.

## 4. Scope

### Included

- Web app internal
- CRM lite
- AI assisted proposal
- Solar sizing calculator
- Pricing engine
- Project kanban
- Dashboard

### Excluded for MVP

- Integrasi penuh IoT inverter
- Mobile app native
- Accounting full ERP
- Procurement advanced

## 5. Stakeholders

| Role | Responsibility |
|---|---|
| Owner | Sponsor, decision maker |
| Sales | Input lead, follow up, proposal |
| Engineer | Review sizing and design |
| Admin Finance | Quotation, invoice coordination |
| Project Manager | Installation tracking |
| Developer | Build platform |

## 6. Timeline

| Phase | Duration |
|---|---:|
| Discovery & Design | 2 weeks |
| MVP Build Sprint 1 | 3 weeks |
| MVP Build Sprint 2 | 3 weeks |
| Testing & Pilot | 2 weeks |
| Launch | 2 weeks |

## 7. Key Risks

- Estimasi AI tidak akurat tanpa rule teknis.
- Data harga material sering berubah.
- Sales tidak disiplin input data.
- Integrasi WhatsApp bisa berubah tergantung provider.

## 8. Governance

Semua output AI untuk pricing, proposal, dan desain teknis harus berstatus Draft sampai disetujui manusia.
