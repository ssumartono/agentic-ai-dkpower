# Operating Model
# DK Power Agentic Energy Business OS

## 1. Daily Sales Routine

1. Check new leads.
2. Review AI summary.
3. Contact high score leads first.
4. Generate recommendation.
5. Request missing data.
6. Generate proposal draft.
7. Send to engineer review.
8. Follow up customer.

## 2. Engineer Routine

1. Review proposal queue.
2. Validate system size.
3. Validate component selection.
4. Check margin and risk.
5. Approve or request revision.

## 3. Owner Routine

1. Review dashboard.
2. Check pipeline value.
3. Check proposal volume.
4. Check won/lost ratio.
5. Check project delay.
6. Review AI exception report.

## 4. Weekly Meeting

Agenda:

- Pipeline review
- Lost reason review
- Project delay review
- Price list update
- AI output correction review
