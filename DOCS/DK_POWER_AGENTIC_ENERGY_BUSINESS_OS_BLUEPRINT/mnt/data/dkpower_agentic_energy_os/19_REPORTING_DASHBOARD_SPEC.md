# Reporting & Dashboard Specification
# DK Power Agentic Energy Business OS

## 1. Owner Dashboard

Metrics:

- Total leads
- Qualified leads
- Proposal sent
- Won deals
- Lost deals
- Pipeline value
- Forecast revenue
- Installed capacity kWp
- Average payback offered
- Average proposal creation time

## 2. Sales Dashboard

Metrics:

- My leads
- Follow up due today
- Proposal pending
- Proposal won
- Lost reason
- Lead source performance

## 3. Engineering Dashboard

Metrics:

- Proposal review queue
- Approved proposals
- Need revision
- Average review time
- Common technical risks

## 4. Project Dashboard

Metrics:

- Active projects
- Delayed projects
- Procurement status
- Installation progress
- Commissioning status

## 5. AI Governance Dashboard

Metrics:

- Number of AI generated proposals
- AI confidence average
- Human correction rate
- Blocked proposals
- Common missing data
